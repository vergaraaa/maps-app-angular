# MapsApp

Don't use AngularCLI direclty, unless env variables are created.

## Steps:

1. Cline .env.template and rename it to .env
2. Fill in env variables
3. Create Angular Envs (optional)

```
npm run envs
```

4. For deployment run:

```
npm run start
```

5. For prod run:

```
npm run build
```
